🛡️ Cybersecurity System Monitor

Python-based cybersecurity monitoring tool for system information, process analysis, network monitoring, and suspicious port detection.
📌 Project Overview

This project is a Python-based cybersecurity monitoring tool designed to collect and display important system and network information in real time.
The tool helps security analysts and system administrators monitor system activity, analyze running processes, inspect network connections, and detect suspicious ports.
